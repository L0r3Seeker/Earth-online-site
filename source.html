关于地球Online

本系统是一款基于现代 Web 技术栈构建的玩家行为度量与成就追踪系统，旨在提供高性能、低延迟的数据可视化方案。

核心特色

  • 海量数据动态渲染，支持 999+ 成就卡片无缝无卡顿加载。
  • 全动态主题响应系统，完美适配深色、波浪与浅色视觉环境。
  • 轻量化前端架构，纯原生高效交互，拒绝冗余依赖。

技术栈架构

纯前端架构

function loadCompleted() {
    try {
        const data = localStorage.getItem(STORAGE_KEY);
        return data ? new Set(JSON.parse(data)) : new Set();
    } catch (e) {
        return new Set();
    }
}

function saveCompleted(ids) {
    localStorage.setItem(STORAGE_KEY, JSON.stringify([...ids]));
}

function toggleAchievement(id) {
    const completed = loadCompleted();
    if (completed.has(id)) {
        completed.delete(id);
    } else {
        completed.add(id);
    }
    saveCompleted(completed);
    return completed.has(id);
}

关键代码解析

1. 本地数据持久化

架构提示

使用 localStorage 存储用户已解锁的成就 ID，数据完全保存在用户本地浏览器中，换设备或清除浏览器数据会丢失。采用 Set 数据结构实现 O(1) 时间复杂度的存在性校验。

4. SVG 图标生成系统

const SVG_COLORS = {
    gold:   ["#FFD700", "#FFA500", "#B8860B"],
    silver: ["#C0C0C0", "#A9A9A9", "#808080"],
    copper: ["#CD7F32", "#B87333", "#8B4513"],
};

function generateSVG(iconName, rarity = 'silver') {
    const [c1, c2, c3] = SVG_COLORS[rarity] || SVG_COLORS.silver;
    const pattern = SVG_PATTERNS[iconName] || SVG_PATTERNS.star;
    return pattern.replace(/{c1}/g, c1)
                  .replace(/{c2}/g, c2)
                  .replace(/{c3}/g, c3);
}
架构提示

通过模板占位符替换颜色，实现"一套图案 + 多套配色"的灵活组合，确保图标系统与 .exe 版本完全一致。
